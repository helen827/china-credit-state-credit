## 核心内容

### 演讲主题
中国经济的国家信用：信用主体切换与产业趋势分析

### 核心论点
1. **四代信用主体演变**：国家信用 → 外贸企业信用 → 地方政府与房地产信用 → 中央财政与科技产业信用
2. **房地产退潮的本质**：信用主体的切换，不是简单的周期波动
3. **新信用主体的四大能力标准**：提高生产率、形成真实现金流、承接国家战略资源、进入全球市场
4. **K型分化**：财富机会从普遍性变为筛选性
5. **经济周期的新理解**：从市场产物到国家产物

#### 2. 高新技术企业税收优惠

#### 3. IPO高技术企业占比
- 2024年A股IPO总数：**100家**（较2023年313家大幅下降）
- 科创板+创业板占比：**~80%**（科创板15家+创业板38家）

## 技术说明

- **模板**：基于 Cartesian 模板（beautiful-html-templates）
- **图表**：Chart.js
- **字体**：Playfair Display + Inter
- **交互**：键盘导航（左右箭头）、触摸滑动、导航点
